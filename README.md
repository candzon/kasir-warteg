
# KASIR WARTEG

Projek ini dibuat untuk memenuhi syarat UAS dengan menggunakan bahasa pemrograman VB.NET




## Features

- Login
- MDI (Multiple Document Interface)
- Fullscreen mode
- Auto SUM


## Installation

Install my-project with git

```bash
  https://github.com/candzon/kasir-warteg.git
  cd kasir-warteg
```
